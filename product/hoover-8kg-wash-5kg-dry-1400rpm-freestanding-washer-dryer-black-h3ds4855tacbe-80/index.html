Hoover 8kg Wash 5kg Dry 1400rpm Freestanding Washer Dryer Black - H3DS4855TACBE-80

 

This Hoover washer dryer has an 8kg wash and a 5kg dry load capacity, as well as a 1400 spin speed. KG Mode will weigh the load during the first four minutes and adjusts the length of the programme and the water consumption accordingly. The clever Sensor Dry function allows you to choose the level of dryness you require from three options, and the machine will automatically switch off when it senses that this level has been reached, saving you time and energy. You can also use the Timed Drying function, which will let you set the cycle for a specific amount of time. NFC technology allows you to connect to the machine via an app on smartphones, so you can choose and monitor cycles and run diagnostic checks.
